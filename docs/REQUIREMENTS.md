# 애플리케이션 요구 사항

## 1. 화면 구성

- [ ] 1.1. 화면은 3개의 영역으로 구성한다.
  - [ ] 1.1.1. 상단 영역은 여러가지 기능을 제공하는 아이콘 버튼들이 왼쪽에, 가운데에는 검색창, 그리고 오른쪽에는 설정한 데드라인과 날씨가 표시된다.
  - [ ] 1.1.2. 중단 영역은 왼쪽에 즐겨찾기 목록, 가운데에서 상단에는 시간과 인사말, 가운데에서 하단에는 할 일 입력창, 오른쪽에는 할 일 목록 표시된다.
  - [ ] 1.1.3. 하단 영역은 인용구가 랜덤하게 표시된다.
- [ ] 1.2. 화면은 데스크탑, 랩탑, 태블릿 PC, 모바일 환경에 최적화된다.
- [ ] 1.3. 화면은 가로 방향으로 화면이 넓어질 때 중단 영역이 확대되고 좁아질 때 중단 영역이 축소된다.
- [ ] 1.4. 화면의 가로 영역이 태블릿 PC 이하인 환경에서는 상단 영역에 검색창 아이콘과 날씨 아이콘만 표시되고, 여러가지 기능을 제공하는 아이콘 버튼은 사이드바에 숨겨진다.
  - [ ] 1.4.1. 사이드바는 토글 버튼을 클릭하면 나타나고 다시 클릭하면 사라진다.
  - [ ] 1.4.2. 사이드바는 여러가지 기능을 제공하는 아이콘 버튼들이 표시된다.
  - [ ] 1.4.3. 사이드바는 화면의 왼쪽에서 오른쪽으로 나타나고 사라진다.
  - [ ] 1.4.4. 사이드바는 닫기 버튼을 클릭하거나 사이드바 외부를 클릭하면 사라진다.
  - [ ] 1.4.5. 검색 아이콘을 클릭하면 검색창이 나타나고, 다시 클릭하면 사라진다.

## 2. 로그인

- [ ] 2.1. 가운데 나타나는 입력란에 이름을 입력하고 엔터를 누르면 인사말이 나타난다.
- [ ] 2.2. 이름을 입력하지 않고 엔터를 누르면 `이름을 입력해주세요.`라는 메시지가 나타난다.
- [ ] 2.3. 이름을 입력하고 엔터를 누르면 입력란이 사라지고 인사말이 나타난다.
- [ ] 2.4. 인사말은 시간대 별로 `Good Morning, {이름}`, `Good Afternoon, {이름}`, `Good Evening, {이름}`, `Good Night, {이름}`으로 나타난다.

## 3. 할 일 관리

- [ ] 3.1. 할 일을 입력하고 엔터를 누르면 화면의 중단 영역 오른쪽에 할 일이 추가된다.
- [ ] 3.2. 할 일을 클릭하면 완료된 할 일로 표시된다.
- [ ] 3.3. 할 일을 삭제하면 할 일이 삭제된다.
- [ ] 3.4. 할 일을 완료한 후 삭제하면 완료된 할 일이 삭제된다.
- [ ] 3.5. 할 일을 완료한 후 삭제하지 않으면 완료된 할 일이 유지된다.
- [ ] 3.6. 할 일을 완료한 후 다시 클릭하면 완료되지 않은 할 일로 표시된다.
- [ ] 3.7. 할 일의 완료 여부는 체크박스로 표시된다.
- [ ] 3.8. 할 일은 입력한 순서대로 표시된다.
- [ ] 3.9. 할 일을 입력하면 스케줄에 해당 날짜의 할 일로 추가된다.

## 4. 인용구

- [ ] 4.1. 하단 영역에 인용구가 랜덤하게 표시된다.
- [ ] 4.2. 인용구는 테마가 있으며, 설정에서 선택한 테마에 따라 인용구가 표시된다.
- [ ] 4.3. 인용구는 `인용구 내용 - 인용구 작성자`로 표시된다.

## 5. 날씨

- [ ] 5.1. 상단 영역에 오른쪽에 날씨가 표시된다.
- [ ] 5.2. 날씨는 현재 위치의 날씨 정보를 표시한다.

## 6. 즐겨찾기

- [ ] 6.1. 중단 영역 왼쪽에 즐겨찾기 목록이 표시된다.
- [ ] 6.2. 즐겨찾기 목록은 네모 모양의 아이콘(`해당 사이트의 탭 아이콘`)으로 표시되며, 아이콘 하단에 입력한 이름이 표시된다.
  - [ ] 6.2.1. 즐겨찾기 목록은 2x6 그리드로 표시되며, 하단에 페이지네이션이 동그란 아이콘으로 표시된다.
  - [ ] 6.2.1. 특정 페이지네이션 아이콘을 클릭하면 해당 페이지의 즐겨찾기 목록으로 슬라이드되며, 아이콘은 활성화된다.
- [ ] 6.3. 즐겨찾기 목록은 아이콘을 클릭하면 해당 사이트로 이동한다.
- [ ] 6.4. 즐겨찾기 영역에 마우스를 올리면 영역의 오른쪽 위에 케밥 메뉴 아이콘이 표시되며, 클릭하면 `사이트 추가`, `폴더 생성` 메뉴 버튼이 드롭다운으로 표시된다.
  - [ ] 6.4.1. `사이트 추가` 메뉴 버튼을 클릭하면 즐겨찾기를 추가할 수 있는 모달이 나타나며, 이름과 URL을 입력한 뒤 `저장` 버튼을 클릭하면 즐겨찾기 목록에 추가된다.
  - [ ] 6.4.2. `폴더 생성` 메뉴 버튼을 클릭하면 기존에 추가된 즐겨찾기 목록을 선택할 수 있으며, 즐겨찾기 목록을 선택한 뒤 `폴더 생성` 버튼을 클릭하면 폴더 이름을 입력할 수 있는 모달이 나타난다. 폴더 이름을 입력한 뒤 `저장` 버튼을 클릭하면 폴더가 생성되고 선택한 즐겨찾기 목록이 폴더에 추가된다.
- [ ] 6.5. 즐겨찾기 목록의 폴더를 클릭하면 모달이 나타나며, 해당 폴더에 추가된 즐겨찾기 목록이 표시된다.
- [ ] 6.6. 즐겨찾기 아이콘을 드래그하여 순서를 변경할 수 있다.
- [ ] 6.7. 즐겨찾기 목록에 마우스를 올리면 아이콘의 오른쪽 위에 수정 아이콘이 표시되며, 클릭하면 `사이트 수정`과 `삭제` 버튼이 있는 드롭다운 메뉴가 표시된다.
  - [ ] 6.7.1. `사이트 수정` 버튼을 클릭하면 즐겨찾기를 수정할 수 있는 모달이 나타나며, 이름과 URL을 수정한 뒤 `저장` 버튼을 클릭하면 즐겨찾기 목록이 수정된다.
  - [ ] 6.7.2. `삭제` 버튼을 클릭하면 즐겨찾기 목록이 삭제된다.

## 7. 데드라인 설정

- [ ] 7.1. 상단 영역에 오른쪽에 데드라인이 표시된다.
- [ ] 7.2. 데드라인에 마우스를 올리면 오른쪽 위에 케밥 아이콘이 표시되고, 아이콘을 클릭하면 `데드라인 추가`, `데드라인 수정`, `데드라인 삭제` 버튼이 드롭다운 메뉴로 표시된다.
  - [ ] 7.2.1. `데드라인 추가` 버튼을 클릭하면 데드라인을 추가할 수 있는 모달이 나타나며, 날짜와 시간을 입력한 뒤 `저장` 버튼을 클릭하면 데드라인이 추가된다.
  - [ ] 7.2.2. `데드라인 수정` 버튼을 클릭하면 데드라인을 수정할 수 있는 모달이 나타나며, 날짜와 시간을 수정한 뒤 `저장` 버튼을 클릭하면 데드라인이 수정된다.
  - [ ] 7.2.3. `데드라인 삭제` 버튼을 클릭하면 데드라인이 삭제된다.
- [ ] 7.3. 데드라인은 설정한 날짜와 시간이 표시된다.
- [ ] 7.4. 데드라인은 설정한 날짜와 시간이 지나면 알림이 울리고, 알림에는 `데드라인이 지났습니다.`라는 메시지가 표시된다.

## 8. 랜덤 배경화면

- [ ] 8.1. 화면의 배경화면은 랜덤하게 변경된다.
- [ ] 8.2. 배경화면은 사용자가 설정할 수 있다.

## 9. 기능

### 9.1. 검색

- [ ] 9.1.1. 상단 영역의 검색창에는 검색어를 입력할 수 있다.
- [ ] 9.1.2. 검색어를 입력하고 엔터를 누르면 새 탭으로 검색 결과 페이지가 열린다.
- [ ] 9.1.3. 검색 결과 페이지는 구글 검색 결과 페이지이다.

### 9.2. 스케줄

- [ ] 9.2.1. 스케줄 버튼을 클릭하면 스케줄을 확인할 수 있는 모달이 나타난다.
- [ ] 9.2.2. 스케줄 모달에는 달력이 표시되고, 해당 날짜에 할 일을 입력할 수 있다.
- [ ] 9.2.3. 스케줄 모달에는 달력이 표시되고, 해당 날짜의 할 일을 확인할 수 있다.
- [ ] 9.2.4. 스케줄 모달에서 할 일을 클릭하면 할 일을 수정할 수 있다.
- [ ] 9.2.5. 스케줄 모달에서 할 일을 삭제하면 할 일이 삭제된다.

### 9.3. 타이머

- [ ] 9.3.1. 타이머 버튼을 클릭하면 타이머를 설정할 수 있는 모달이 나타난다.
- [ ] 9.3.2. 타이머 모달에는 타이머 시간과 휴식 시간을 입력할 수 있는 입력창이 표시된다.
- [ ] 9.3.3. 타이머 모달에는 시작 버튼과 초기화 버튼이 표시된다.
- [ ] 9.3.4. 타이머 모달에는 타이머가 종료되면 알림이 울리고, 알림에는 `타이머 종료`라는 메시지가 표시된다.

### 9.4. 음악 플레이어

- [ ] 9.4.1. 음악 플레이어 버튼을 클릭하면 음악 플레이어를 확인할 수 있는 모달이 나타난다.
- [ ] 9.4.2. 음악 플레이어 모달에는 음악 목록이 표시된다.
- [ ] 9.4.3. 음악 플레이어 모달에는 음악을 재생, 일시정지, 다음 곡, 이전 곡을 할 수 있는 버튼이 표시된다.
- [ ] 9.4.4. 음악 플레이어 모달에는 음악을 들을 수 있는 플레이어가 표시된다.

### 9.5. 설정

- [ ] 9.5.1. 설정 버튼을 클릭하면 설정을 확인할 수 있는 모달이 나타난다.
- [ ] 9.5.2. 설정 모달에는 인용구 테마, 위치, 시간대, 랜덤 배경화면을 설정할 수 있는 입력창이 표시된다.
- [ ] 9.5.3. 설정 모달에는 저장 버튼이 표시된다.
- [ ] 9.5.4. 설정 모달에는 저장 버튼을 클릭하면 설정이 저장된다.
